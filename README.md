# My Card

Projeto bem simples contendo meu card.
## Como ter uma copia?

Faça do download do repositório pelo ``git clone``, e siga o passo a passo:

```
// Entre na pasta
cd my-card

// Apague a pasta .git
rm -rf .git

// Instale as dependências
npm install
```
